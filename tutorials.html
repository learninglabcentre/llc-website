This is your path to a career in UX design. In this course, you’ll learn in-demand skills that will help you understand key skills about User Experience design. No degree or experience required.

The main focus of this course is to give you industry experience. What it looks like when you work in the corporate world being a UX Designer.

Learn UX foundations and gain experience with the design process, including how to build wireframes and prototypes and conduct user research to test your designs.

Total Duration 140 hours
3 hours per day (Note: Considering 2 days a week it takes approx 5 months to complete the course.)

  1. Introduction to UX and UI
    • Definition and differences between UX and UI
    • Why they matter in product design
    • Overview of the design process
  2. User Research
    • Understanding user needs and behavior
    • Conducting user interviews and surveys
    • Creating user personas
  3. Information Architecture and Navigation:
    • Developing site maps and wireframes
    • Creating effective navigation and labeling systems
    • Understanding hierarchy and organization
  4. Interaction Design:
    • Designing user flows interfaces
    • Designing forms and data input
    • Understanding gestures and interactions
  5. Visual Design:
    • Basic design principles such as color theory and typography
    • Creating layouts and visual hierarchy
    • Creating icons and graphics
  6. Usability Testing:
    • Planning and conducting usability tests
    • Analyzing and interpreting results
    • Iterating on designs
  7. Design Tools and Techniques:
    • Introduction to popular design tools such as Sketch, Figma, or Adobe XD
    • Creating high-fidelity prototypes
    • Using design frameworks and templates
  8. Mobile and Responsive Design:
    • Designing for mobile devices
    • Understanding responsive design
    • Creating mobile-first designs
  9. Career Opportunities in UX and UI:
    • Overview of different roles in UX and UI
    • Skills and qualifications required for UX and UI roles
    • Building a career in UX and UI
